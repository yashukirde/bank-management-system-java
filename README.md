# Bank Management System (Java)

This repository contains a **Bank Management System developed in Java** as part of academic coursework.
The project demonstrates basic banking operations using **Object-Oriented Programming (OOP)** concepts.

## Project Structure

```
src/
 └── bms/
      ├── Account.java
      ├── AccountManager.java
      ├── BankingApp.java
      └── User.java
```

### File Description

* **Account.java**
  Represents a bank account and handles account-related information.

* **AccountManager.java**
  Manages banking operations such as account creation, deposit, withdrawal, and balance management.

* **BankingApp.java**
  Main application file that runs the banking system and handles user interaction.

* **User.java**
  Represents user details associated with bank accounts.

## Features

* Create bank accounts
* Deposit money
* Withdraw money
* Manage account details
* Basic banking operations simulation

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Eclipse IDE

## Purpose

This project was developed to practice and understand fundamental **Java programming concepts and banking system logic** through a simple console-based application.
